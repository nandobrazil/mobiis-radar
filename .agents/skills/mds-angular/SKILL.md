# mds-components

```
$ npx skills add https://github.com/joseorlandosousa/mds-skills --skill mds-angular
```

This skill provides documentation and examples for MDS-Angular (Mobiis Design System) with accessible, styled, and interactive components. Use this skill to understand APIs, behaviors, Inputs/Outputs, and best practices for implementing Angular UI interfaces.
MDS Angular is a collection of accessible, styled, and highly interactive components that implement the standard Mobiis Design System patterns. The components handle styling, common behaviors, and provide robust Angular APIs (Inputs/Outputs) for seamless integration.

Install via npm install (internal package):
```
npm install @fretefy/mds-angular
```

## When to Apply
Use this skill when:
- Building standard Mobiis pages and interfaces in Angular
- Needing pre-styled, design-system compliant UI components
- Implementing complex forms, navigation, or data visualization patterns
- Following the standard architectural patterns for layout and interactions
- Implementing custom dashboards or delivery lists with MDS guidelines

Do NOT use this skill when:
- You need completely custom or unstyled elements contrary to MDS standards
- Building prototypes outside the Mobiis ecosystem
- Using Angular Material directly (MDS Components should be your primary UI library)
