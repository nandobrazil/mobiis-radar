# Tooltip

## Description
A pure text tiny floating bubble hovering immediately over elements, shown specifically on user mouse-hover interactions indicating brief hints.

## When to Apply
Use this component when:
- Clarifying an icon-only button meaning
- Spelling out truncating ellipsized text elements entirely

Do NOT use this component when:
- Housing interactive content or forms inside the tooltip bubble itself

## API Reference (Inputs/Outputs)

### Inputs
