# Section Header

## Description
A distinct title typography block indicating the beginning of a sub-section inside a complex form or lengthy informational page.

## When to Apply
Use this component when:
- Breaking up long complex forms into "Personal Info", "Address Data", "Billing"
- Grouping metrics together contextually

Do NOT use this component when:
- As a replacement for Page Header or primary document boundaries
